# Notes App

A full CRUD notes application built with React.

## ✨ Features

* Add, edit and delete notes
* Edit cancel functionality
* Local storage persistence
* Clean UI interactions

## 🛠️ Tech Used

* React (Hooks)
* JavaScript
* Local Storage API

## 📚 What I Learned

* CRUD operations in React
* Controlled components
* Managing state transitions
