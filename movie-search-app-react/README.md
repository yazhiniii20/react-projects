# Movie Search App

A React application that allows users to search and explore movies using an external API.

## ✨ Features

* Movie search functionality
* API integration
* Dynamic rendering of results
* Component-based architecture

## 🛠️ Tech Used

* React
* Fetch API / Axios
* CSS

## 📚 What I Learned

* Working with APIs
* Handling async data
* Conditional rendering
